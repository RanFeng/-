# 中国象棋
简单的两人对战中国象棋游戏
--------
请在 python2.7 环境下使用,界面很丑，期待改进

-----
```
python2.7 chess.py
```

usage
=======
首先运行服务器端
```python2.7 udp_server.py``` </br>
first, playerA 输入昵称,然后 Link The Server,</br>
then , playerB 输入昵称,Link The Server,</br>
这时B能看到A，而A不能看到B，然后，B邀请A（双击列表中的A），A接受邀请，游戏开始。</br>
红方先动。</br>
整个过程就是这样，具体可以看代码。</br>
